# FRONTEND REQUIREMENTS

## Stack
- Next.js 14 + TypeScript
- Tailwind CSS + Shadcn/ui
- React Query + Zustand

## Pages
- /login - Login
- /register - Register
- /dashboard - Overview
- /jobs - Jobs list
- /jobs/[id] - Job details
- /candidates/[id] - Candidate profile

## Key Features
1. Auth flow
2. Job CRUD
3. CV upload (drag & drop)
4. Candidate scoring display
5. Status management
